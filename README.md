# Simple-File-Manger
README: Simple Java File Manager (GUI)This repository contains a desktop-based File Management Application built using Java Swing. It allows users to interact with their local file system through a graphical interface to create, open, edit, and save text files.### Project FeaturesVisual File Selection: Uses JFileChooser to provide standard "Open" and "Save" dialog boxes, eliminating the need to manually type file paths.Multi-line Editor: Features a JTextArea wrapped in a JScrollPane, allowing for the viewing and editing of large text documents.Stream-Based I/O: Implements BufferedReader and BufferedWriter for efficient, high-performance reading and writing operations.User Feedback: Utilizes JOptionPane to provide clear success or error messages during file operations.Clean Layout: Organized using BorderLayout and FlowLayout to ensure a responsive and intuitive user experience.### How to Run the Program1. PrerequisitesJDK 8 or higher installed.A Java-compatible IDE (IntelliJ, Eclipse, NetBeans) or a command-line interface.2. CompilationOpen your terminal and compile the Java file:Bashjavac FileManagerGUI.java
3. ExecutionRun the application:Bashjava FileManagerGUI
### Core Logic BreakdownActionLogic ComponentDescriptionOpenshowOpenDialogOpens a file browser to select an existing .txt file.ReadreadLine()Reads the file line-by-line and appends it to the text area.SaveshowSaveDialogOpens a file browser to choose a save location and filename.Writewriter.write()Takes all text currently in the editor and writes it to the disk.ClearsetText("")Clears the editor to start a fresh document.### Application GuideNew File: Click "New/Clear" to wipe the editor and start typing fresh content.Opening: Click "Open File" and navigate to any text file on your computer to load its content.Saving: After typing or editing, click "Save/Write". You can either overwrite an existing file or type a new name to create one.### Future EnhancementsFile Filters: Restrict the JFileChooser to only show .txt or .java files.Auto-Save: Implement a timer that automatically saves the current progress to a temporary file.Word Count: Add a status bar at the bottom to show the current character and word count.
